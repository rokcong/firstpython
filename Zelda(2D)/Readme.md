# 💻 How to play (Windows)
### 조작키
```
⬆️ = 전진
⬇️ = 후진
➡️ = 오른쪽
⬅️ = 왼쪽
스페이스바 = 물리 공격
Lctrl = 마법 공격
Q : 무기 변경
E : 마법 변경
```
### 📦 관련 모듈 설치
```
pip install pygame
pip install pip
```
